# Code of Conduct

## Our Standards

Participants are expected to:

- ebrace the slop, deslopify over time
- focus feedback on code and behavior, not people
- assume good intent while asking clarifying questions

Unacceptable behavior includes:

- taking yourself or the project too seriously

## Scope

This Code of Conduct applies to repository issues, pull requests, discussions, and other project collaboration spaces.

## Enforcement

Project maintainers are responsible for interpreting and enforcing this policy.
Maintainers may remove comments, close threads, or restrict participation for violations.

## Reporting

Report serious or repeated conduct concerns privately to maintainers.
If a private contact path is not obvious, open a minimal issue requesting one.
